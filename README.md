# Training Planner
# Автор: Артем Хабаров

# Описание проекта
Training Planner — это простое десктопное приложение с графическим интерфейсом (GUI), написанное на языке Python. Оно предназначено для ведения личного дневника тренировок.

# Приложение позволяет:

Добавлять новые записи о тренировках (дата, тип и длительность).
Визуализировать историю занятий в виде таблицы.
Фильтровать записи по типу тренировки или дате.
Сохранять все данные в локальный файл trainings.json и загружать их обратно.
Проверять корректность вводимых пользователем данных.
Проект настроен для работы с системой контроля версий Git.

# Технологии
Язык: Python 3.x.
Библиотека GUI: Tkinter (встроена в Python).
Формат данных: JSON.
Контроль версий: Git.
Инструкция по запуску
Убедитесь, что на вашем компьютере установлен Python 3.6 или новее.
Скопируйте файл main.py в папку проекта.
Откройте терминал (командную строку) в этой папке.
Запустите приложение командой:
bash
Копировать
python main.py
Графическое окно приложения откроется автоматически.
# Примеры использования (Тестирование)
Для проверки работоспособности приложения выполните следующие шаги:

Тест 1: Добавление тренировки и валидация даты
Введите в поля: Дата — 2026-05-04, Тип — Бег, Длительность — 30.
Нажмите кнопку «Добавить тренировку».
Ожидаемый результат: Новая строка появилась в таблице.
Теперь введите неверную дату: 4-5-26.
Нажмите кнопку «Добавить тренировку».
Ожидаемый результат: Появится окно с ошибкой «Неверный формат даты. Используйте ГГГГ-ММ-ДД».
Введите отрицательную длительность: -15.
Нажмите кнопку «Добавить тренировку».
Ожидаемый результат: Появится окно с ошибкой «Длительность должна быть положительным числом».
Тест 2: Фильтрация данных
Добавьте несколько тренировок с разными типами (например: «Бег», «Силовая», «Плавание»).
В поле фильтра «По типу» введите: бег.
Нажмите кнопку «Применить фильтр».
Ожидаемый результат: В таблице остались только строки с типом «Бег».
Нажмите кнопку «Сбросить фильтр».
Ожидаемый результат: Все записи снова отображаются в таблице.
Тест 3: Сохранение и загрузка данных
Добавьте 2–3 тренировки.
Нажмите кнопку «Сохранить в JSON».
Ожидаемый результат: В папке проекта появится файл trainings.json. Появится сообщение об успехе.
Выделите все строки в таблице и нажмите клавишу Delete, чтобы очистить её.
Нажмите кнопку «Загрузить из JSON».
Ожидаемый результат: Все ранее сохраненные тренировки снова появились в таблице.
