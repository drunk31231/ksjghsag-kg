# Training Planner

**Автор:** Хабаров Артем

**Описание:**
Приложение для планирования тренировок с возможностью добавления, фильтрации и сохранения данных в JSON.

**Примеры использования:**
1. Запустите main.py.
2. Введите дату (ГГГГ-ММ-ДД), тип тренировки и длительность.
3. Нажмите «Добавить тренировку».
4. Используйте фильтрацию по типу и дате.
5. Сохраняйте и загружайте данные через кнопки.
6. Все изменения отслеживаются через Git.
