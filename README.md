# Training Planner
# Автор: Артем Хабаров

# Описание проекта
Training Planner — это простое десктопное приложение с графическим интерфейсом (GUI), написанное на языке Python. Оно предназначено для ведения личного дневника тренировок.

# Примеры использования

Ведение дневника: 
Вы можете ежедневно записывать свои тренировки, указывая тип (например, «Кардио», «Силовая» или «Плавание»), дату и продолжительность, чтобы отслеживать свой прогресс.

Быстрый поиск: 
Если вы хотите вспомнить, когда в последний раз ходили на «Йогу» или сколько времени уделили «Бегу» на прошлой неделе, просто введите ключевое слово в фильтр и нажмите «Применить».

Сохранение данных: 
Все ваши записи можно сохранить в файл одним нажатием кнопки, чтобы они не потерялись при закрытии программы.

Анализ нагрузки: 
Загрузив данные из файла, вы можете легко оценить свою активность за месяц, просто просмотрев накопленные записи в таблице.
