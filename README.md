# Training Planner
## Автор: [Хабаров Артем]

## Описание проекта
Training Planner — это простое десктопное приложение с графическим интерфейсом (GUI), написанное на языке Python. Оно предназначено для ведения личного дневника тренировок.

## Приложение позволяет:

1) Добавлять новые записи о тренировках (дата, тип, длительность).
2) Визуализировать историю занятий в виде таблицы.
3) Фильтровать записи по типу тренировки или дате.
4) Сохранять все данные в локальный файл trainings.json и загружать их обратно.
5) Проверять корректность вводимых пользователем данных.
6) Проект настроен для работы с системой контроля версий Git.


## Тест на добавление тренировки:
Введите в поля: Дата — 2026-05-04, Тип — Бег, Длительность — 30.
Нажмите кнопку «Добавить тренировку».
Ожидаемый результат: Новая строка появилась в таблице.
# Тест на валидацию (ошибка даты):
Введите в поле Дата некорректное значение, например: 04-05-2026.
Нажмите кнопку «Добавить тренировку».
Ожидаемый результат: Появится окно с ошибкой «Неверный формат даты. Используйте ГГГГ-ММ-ДД».
# Тест на фильтрацию:
Добавьте несколько тренировок с разными типами (например, «Бег», «Силовая», «Плавание»).
В поле фильтра «По типу» введите: Бег.
Нажмите кнопку «Применить фильтр».
Ожидаемый результат: В таблице остались только строки, где тип тренировки содержит слово «Бег».
# Тест на сохранение и загрузку:
Добавьте 2-3 тренировки.
Нажмите кнопку «Сохранить в JSON».
Ожидаемый результат: В папке проекта появится файл trainings.json. Закройте приложение.
Удалите все строки из таблицы (выделите и нажмите Delete) или просто запустите приложение заново.
Нажмите кнопку «Загрузить из JSON».
Ожидаемый результат: Все ранее сохраненные тренировки снова появились в таблице.
