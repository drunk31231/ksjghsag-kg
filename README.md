# Training Planner
# Автор: Артем Хабаров

# Описание проекта
Training Planner — это простое десктопное приложение с графическим интерфейсом (GUI), написанное на языке Python. Оно предназначено для ведения личного дневника тренировок.

# Примеры использования
Запуск дневника
Откройте приложение. Перед вами пустая таблица, готовая к заполнению.

Добавление новой тренировкиДопустим, вы только что закончили утреннюю пробежку.
В поле «Дата» введите: 2026-05-04.
В поле «Тип тренировки» введите: Бег.
В поле «Длительность» введите: 45.
Нажмите кнопку «Добавить тренировку». Запись мгновенно появится в таблице.

Фильтрация записей
Через неделю вы хотите вспомнить, как часто вы занимались йогой.
Введите йога в поле фильтра «По типу».
Нажмите «Применить фильтр». В таблице отобразятся только ваши занятия йогой.
Чтобы вернуть полный список, нажмите кнопку «Сбросить фильтр».

Сохранение прогресса
В конце недели вы хотите сохранить все свои результаты, чтобы не потерять их.
Просто нажмите кнопку «Сохранить в JSON». Приложение создаст файл trainings.json со всеми вашими данными.

Продолжение работы на другом устройстве (или после перезапуска)
Если вы закрыли программу и открыли её снова, ваши данные исчезнут из памяти приложения.
Чтобы их вернуть, нажмите кнопку «Загрузить из JSON». Все ваши прошлые тренировки снова появятся в таблице.
